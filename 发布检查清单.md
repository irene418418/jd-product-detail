# 发布检查清单

在将项目发布到 GitHub 之前，请确保完成以下检查项。

## 📋 代码准备

- [ ] 所有功能已完成并测试
- [ ] 代码已通过 lint 检查：`npm run lint`
- [ ] 项目可以成功构建：`npm run build`
- [ ] 本地预览正常：`npm run preview`
- [ ] 移除所有调试代码和 console.log
- [ ] 移除所有 TODO 注释或创建对应 Issue

## 📝 文档更新

- [ ] README.md 已更新
  - [ ] 项目描述准确
  - [ ] 功能列表完整
  - [ ] 安装步骤清晰
  - [ ] 使用说明详细
- [ ] 更新 package.json 中的信息
  - [ ] 版本号
  - [ ] 描述
  - [ ] 作者
  - [ ] 仓库地址
  - [ ] 主页地址
- [ ] LICENSE 文件已添加
- [ ] CONTRIBUTING.md 已创建
- [ ] 部署指南已完善

## 🔧 配置文件

- [ ] .gitignore 包含所有必要的忽略项
- [ ] .gitattributes 已配置
- [ ] vite.config.ts 的 base 路径正确
- [ ] GitHub Actions 工作流配置正确
- [ ] 环境变量已配置（如需要）

## 🎨 资源文件

- [ ] 项目图标/Logo 已添加
- [ ] 截图已准备（可选）
- [ ] 演示 GIF 已准备（可选）
- [ ] 所有图片已优化

## 🔐 安全检查

- [ ] 没有硬编码的密钥或敏感信息
- [ ] 没有提交 .env 文件
- [ ] 依赖包没有已知的安全漏洞：`npm audit`
- [ ] API 端点使用环境变量配置

## 🌐 GitHub 仓库设置

- [ ] 仓库描述已填写
- [ ] 仓库主题/标签已添加
- [ ] README 在仓库首页正确显示
- [ ] GitHub Pages 已启用
  - [ ] Source 设置为 GitHub Actions
  - [ ] 自定义域名已配置（如需要）
- [ ] Issue 模板已添加
- [ ] PR 模板已添加

## 🚀 部署测试

- [ ] 推送代码到 main 分支
- [ ] GitHub Actions 工作流运行成功
- [ ] GitHub Pages 部署成功
- [ ] 在线演示可以正常访问
- [ ] 所有功能在线上环境正常工作
- [ ] 在不同浏览器测试
  - [ ] Chrome
  - [ ] Firefox
  - [ ] Safari
  - [ ] Edge

## 📢 发布准备

- [ ] 更新 README 中的在线演示链接
- [ ] 更新 README 中的徽章链接
- [ ] 创建 Release 标签
- [ ] 编写 Release Notes
- [ ] 在社交媒体分享（可选）

## 🔄 持续维护

- [ ] 设置 GitHub 通知
- [ ] 准备响应 Issue 和 PR
- [ ] 计划定期更新依赖
- [ ] 监控部署状态

## ✅ 最终确认

- [ ] 所有链接都可以正常访问
- [ ] 所有占位符文本已替换
- [ ] 项目可以被其他人 clone 并运行
- [ ] 文档清晰易懂
- [ ] 代码质量达到发布标准

---

## 快速命令

```bash
# 代码检查
npm run lint

# 构建测试
npm run build

# 预览构建
npm run preview

# 安全审计
npm audit

# 查看项目大小
npm run build && du -sh dist

# 提交并推送
git add .
git commit -m "准备发布"
git push origin main
```
